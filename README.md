# CRUD Blog Application

## 📌 Project Overview

This is a Basic CRUD Blog Application with User Authentication developed using PHP and MySQL.

---

## 🚀 Features

- User Registration
- User Login
- Dashboard
- Create Post
- View Posts
- Update Post
- Delete Post
- Logout
- Responsive Bootstrap UI

---

## 🛠 Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- XAMPP

---

## 📂 Project Structure

```
crud_blog/
│
├── auth/
├── config/
├── css/
├── posts/
├── dashboard.php
├── index.php
└── README.md
```

---

## ⚙️ Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project into `htdocs`.
4. Create a database named `blog`.
5. Import/Create the required tables.
6. Open:

```
http://localhost/crud_blog/
```
